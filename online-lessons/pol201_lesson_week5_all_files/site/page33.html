Inference with the Sample Mean: t-distribution vs. z-distribution

Inference with the Sample Mean

Introduction

When conducting statistical inference for the sample mean, we often face a choice between using the t-distribution and the z-distribution. This decision hinges on several key factors, which we will explore in this discussion.

Discussion

Point 1: For doing inference for the sample mean, we almost never know the population standard deviation (\(\sigma\)).

Point 2: The Central Limit Theorem (CLT) applies for a large sample size (N), regardless of:

  • The population distribution of the random variable
  • Whether we know the variance of the population distribution

Point 3: As described in Professor Balkaâ€™s videos (JBStatistics), if our sample is small and the underlying population follows a normal distribution, then in the Z transformation we must replace \(\sigma\) with the sample variance (\(s\)), and we use the t-distribution, which depends on the degrees of freedom, \( df \text{ = } N \text{ - } 1 \). Hence, in this situation, when doing inference of one population mean we use the T-statistic equal to \( T \text{ = } \frac{\bar{X}\text{ - }\mu_0}{s\text{/}\sqrt{N}} \).

Point 4: But to use the t-distribution, the key assumption is that the underlying population is normal. In many cases, we won't be able to assume this is true.

Main idea: This implies that in small samples (where the CLT does not apply), if we are sure the underlying population is not normal, then we lack a way to do inference using conventional tools generally taught at the introductory statistics level. However, most pollsters and political science researchers use relatively large samples. In practice, most researchers use samples of at least 400 cases, well above the \(n \text{ > } 30\) threshold. So, for political science research, we almost always can use the z-distribution by calling upon the CLT.

Second takeaway: Professor Balka is correct that if the underlying population can surely be assumed to be normal (for example, we know that height and test scores are mostly normal), then the t-distribution will always be correct, as we don't need to rely on the CLT.

Third takeaway: When is my sample size large? This is very subjective. For example, for proportions, the book says that the CLT applies when \(p \text{ $\cdot$ } n \text{ > } 10\) and \(\text{(}1 \text{ - } p\text{)} \text{ $\cdot$ } n \text{ > } 10\) . Others suggest a threshold of 5 for both the positive and negative cases. In truth, unless there is a very rigorous study providing justification, a sample of 50 or 100 might not be large enough to use the CLT depending on the random process we are modeling. Yet, again, to use the t-distribution, we have to assume the underlying random variable, \(X\), follows a normal distribution. This is why, if the sample is small, when doing inference for the population mean, we always use the T distribution and the T statistic.

Conclusion: When we have a small sample size and want to minimize the chance of incorrectly rejecting the null hypothesis (H0) when it is actually true (a Type I error), it's more prudent to assume that the underlying population follows a normal distribution. This approach leads us to use the T-test for hypothesis testing in small samples.
